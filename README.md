# 🍽 Campus Food Ordering Web App
Tools & Technologies: React.js, Node.js, Express.js, MongoDB

Live Demo: https://frontend-ecru-five-46.vercel.app/

  -Developed a full-stack interactive web application that enables students to order food from multiple shops within the NIT Warangal campus, delivered directly to hostel gates.

  -Implemented core features including add/remove items from cart, place orders, and order history to streamline the food ordering experience.

  -Built a responsive front-end using React.js with component-based architecture and dynamic state management.

  -Designed RESTful APIs with Node.js and Express.js, and managed persistent data using MongoDB.

  -Enhanced student convenience and minimized queues at shops, creating a seamless digital ordering experience.

  -Improved operational efficiency for both students and shopkeepers by digitizing the food ordering and delivery process.
